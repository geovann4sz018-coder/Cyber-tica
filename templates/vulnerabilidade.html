Hash MD5 (Exemplo de Hashing Fraco - **NÃO RECOMENDADO!**)

MD5 é um algoritmo de hashing antigo e **altamente vulnerável**. É fácil de "quebrar" (descobrir a senha original a partir do hash) usando tabelas pré-calculadas (rainbow tables).

Hash Gerado: {{ hash_md5 }}

Hash SHA256 (Exemplo de Hashing Mais Forte - **MELHOR, MAS AINDA NÃO IDEAL PARA SENHAS**)

SHA256 é mais seguro que MD5, mas para senhas, algoritmos como **PBKDF2, bcrypt ou scrypt** são preferíveis. Eles são mais lentos intencionalmente, dificultando ataques de força bruta.

Hash Gerado: {{ hash_sha256 }}

No nosso sistema de login, usamos PBKDF2 (implementado pelo Werkzeug), que é muito mais robusto e seguro para armazenar senhas de usuários.
